# Understanding the evolutions of criminality in France between 2016 and 2024

## Description
This project aims to answer key questions about the evolutions of criminality in France : 
  1. Has criminality been on a rise in the past years?
  2. What variables are most closely linked to committing an offense?
  3. For which groups is it possible to see an increase in committing offenses?

For the purpose of our analysis, we will use the database of the statistical service of the French Interior Ministry.

Our analysis will include the following steps : 
Data cleaning and manipulation 
Exploratory data analysis and visualisation 
Interpretation of our findings
